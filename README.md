## File Upload API App

A backend REST API microservices that can be used to accept files (media, pdf, docs, .zpi) submission from user as an input.

The Overview of the app

![Image](./assets/file_upload.png)

### Endpoint

1. Upload files
   `Method: POST`
   `Endpoints: "/upload-images"`

2. Get uploaded file
   `Method: GET - "/"`
